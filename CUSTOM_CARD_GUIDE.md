## ğŸ› **Troubleshooting**

**Q: Background image not showing?**
A: Make sure image file < 5MB, try different format

**Q: Card looks blurry?**
A: We export at 3x scale, should be sharp. Check original image quality.

**Q: Can't upload background?**
A: Check file size, try JPG instead of PNG

**Q: Username not saving?**
A: Clear browser cache, localStorage might be full

**Q: Export taking long?**
A: Large custom backgrounds take time, use presets for faster export
